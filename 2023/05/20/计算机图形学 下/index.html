无偏差的光线传播

双向路径追踪(BDPT)

  • 从摄像机和灯光中追踪子路径
  • 连接两个子路径的端点

如果光的传输在光的一侧是复杂的，则适合。难以实施，速度相当慢。

例如该图中，由于场景本身有较强的光源，其他位置的照明都通过间接光照，从相机出发时，右边墙壁被打到，经过漫反射难以得到从光源来的大量的能量。

Metropolis光线传播 (MLT)

  • 一个马尔科夫链蒙特卡洛（MCMC）的应用，用一些PDF从当前样本跳到下一个样本
  • 非常善于局部探索困难的光路

关键思想：对现有路径进行局部扰动以获得新的路径

优点

  • 对困难的光路有很好的作用，是无偏差的

缺点

  • 难以估计收敛率
  • 不能保证每个像素的收敛率相等。所以，通常会产生 “脏 “的结果
  • 因此，通常不能用于渲染动画

有偏差的光线传播

光子映射

  • 有偏差&两阶段
  • 非常擅长处理Specular-Diffuse-Specular(SDS)路径和生成caustics

实现方法

  • 光子追踪：从光源中发射光子，使其四处反弹，直到打到diffuse平面，然后在漫反射表面记录光子

  • 光子收集：从相机中射出子路径，将它们弹来弹去，直到它们击中漫反射表

  • 计算（Local Density estimation估计）

    • 思路：有更多光子的区域应该更明亮

    • 对于每个阴影点，找到最近的N个光子。取它们所覆盖的表面积

为什么有偏差？

  • Local Density estimation估计 dN / dA != ΔN / ΔA，只有ΔA无限小时才接近一致
  • 但在极限意义上，发出更多的光子 -> 同样的N个光子覆盖了一个较小的ΔA -> ΔA更接近于dA
  • 所以，有偏差，但是一致的!

更容易理解渲染中的偏差

  • 有偏见==模糊不清
  • 一致性==在无限的样本下不模糊

如果ΔA是const range的话永远不会收敛到dA，永远有偏差，所以不采用const range

即时辐射性（IR）

有时也称为多光方法

关键思想：可以将发光的表面作为光源来处理

方法

  • 拍摄光线子路径，并假设每个子路径的终点是一个虚拟点光源（VPL）。
  • 像往常一样使用这些VPLs来渲染场景

优点

  • 速度快，通常对漫反射场景有很好的效果。

缺点

普通微分方程(ODE)

计算粒子随时间变化的位置需要解决一个一阶常微分方程

欧拉法（又称正向欧拉法、显式欧拉法）

  • 简单的迭代方法
  • 非常常用
  • 非常不准确，不准确度随着时间步长 Δt 的增加而增加
  • 经常出现不稳定的情况，会导致模拟发散

在数值积分中，错误会不断积累，欧拉积分尤其糟糕

通过有限差分的数值积分求解会导致两个问题：

错误

  • 每个时间步的误差都会累积。 随着模拟的进行，精度会降低
  • 精度在图形应用中可能并不重要

不稳定性

  • 错误可能会复合，导致模拟发散，即使底层系统没有
  • 缺乏稳定性是模拟中的一个基本问题，不容忽视

对抗不稳定

方法

  • 中点法/修正欧拉：起点和终点的平均速度
  • 自适应步长：递归地比较一步和两个半步，直到误差可以接受
  • 隐式方法：在下一个时间步使用速度（难）
  • 基于位置/Verlet 集成：在时间步后限制粒子的位置和速度
中点法
  • 计算欧拉步骤 (a)
  • 在欧拉步骤 (b) 的中点计算导数
  • 使用中点导数更新位置 (c)

修正欧拉

  • 步骤开始和结束时的平均速度
  • 更好的结果

自适应步长
  • 基于误差估计选择步长的技术
  • 非常实用的技术
  • 但可能需要非常小的步长!

重复进行，直到误差低于阈值：

  • 计算 一个欧拉步骤，大小为T
  • 计算 两个欧拉步长，大小为T/2
  • 计算误差
  • 如果（误差>阈值）减少步长并再试一次

隐式欧拉方法
  • 非正式地称为后向方法
  • 使用未来的导数，用于当前的步骤

  • 解决 和 的非线性问题
  • 使用寻根算法，例如牛顿法。
  • 提供更好的稳定性

如何确定/量化 “稳定性”？

  • 我们使用局部截断误差（每一步）/ 总的累积误差（总体）。
  • 绝对值并不重要，重要的是与步骤有关的阶数。
  • 隐式欧拉的阶数为1，这意味着
    • 局部截断误差： 和
    • 全局截断误差： O(h) (h是步长，即∆t)
  • 对O(h)的理解：如果我们把h减半，我们可以预期误差也会减半

Runge-Kutta Families：用于解决ODE的先进方法系列

  • 特别擅长于处理非线性问题
  • 它的四阶版本是最广泛使用的，又称RK4。

基于位置的/Verlet整合

思想：

  • 在修改过的欧拉正步之后，约束粒子的位置以防止发散的、不稳定的行为
  • 使用约束的位置来计算速度
  • 这两种想法都会耗散能量，稳定下来

优点/缺点

  • 快速和简单
  • 不是基于物理的，会耗散能量（错误）
